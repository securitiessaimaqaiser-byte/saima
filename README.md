# SQS Securities Website Clone

This is a pixel-perfect reverse engineered clone of the SQS Securities website based on video frames provided. Built with React, TypeScript, and Tailwind CSS.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) to view the project.

## Features

*   **Responsive Header**: Includes top bar info, search, login, and navigation dropdowns.
*   **Stock Ticker**: Animated CSS infinite scrolling ticker.
*   **Home Page**: Hero section with negative margin overlapping cards, partnership logos.
*   **Governance Page**: Board of Directors grid layout.
*   **Miscellaneous Page**: FAQ Accordion component.

## Technologies

*   React 18
*   Vite
*   Tailwind CSS
*   Lucide React (Icons)
*   React Router DOM
